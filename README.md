# Real-Time AI Avatar ü§ñ

An interactive, voice-enabled AI Avatar application built with **React**, **Node.js**, **HeyGen Streaming API**, and **Google Gemini Flash**.

This application creates a real-time video conversation experience where users can speak or type to an AI avatar that responds instantly with lip-synced video and intelligent answers.


## ‚ú® Features

* **Real-Time Video Streaming**: Seamless, low-latency avatar video powered by HeyGen.
* **Intelligent Conversations**: Powered by Google's Gemini  Flash model for fast, contextual responses.
* **Voice Interaction**: Built-in "Push-to-Talk" speech recognition (STT) for natural voice chat.
* **Modern UI**: Professional dark-themed interface built with **Tailwind CSS v4** and **Lucide Icons**.
* **Smart Navigation**: Multi-page routing with safety guards to prevent accidental session disconnection.
* **Dual Input**: Supports both voice and text input.
